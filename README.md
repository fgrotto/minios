# Minios

Learning project to dive into RISC-V ISA and OS development.

## Requirements

- Small OS for 32-bit RISC-V ISA (we will extend it for 64-bit)
- QEMU support

## Setup

To run the initial setup:
```bash
./setup.sh 
```

To run the OS in QEMU. CTRL-A C then q to quit:
```bash
./run.sh
```
